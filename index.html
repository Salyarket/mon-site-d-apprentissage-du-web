Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le web est un système d'information mondial qui permet l'accès à des ressources, telles que des pages web, via internet, en utilisant des navigateurs. Il repose sur des technologies comme le HTTP et le HTML pour faciliter la navigation et l'interaction avec les contenus en ligne.

Internet

Internet est un réseau mondial de communication qui relie des milliards d'appareils, permettant l'échange d'informations et l'accès à des services en ligne. Il repose sur une infrastructure de serveurs, de câbles et de protocoles qui facilitent la transmission de données à travers le monde.

Cloud

Le cloud est un système de stockage et de traitement de données en ligne, permettant d'accéder à des fichiers et applications via internet sans avoir besoin de les conserver sur des appareils locaux. Il offre des services de sauvegarde, de collaboration et d'hébergement, tout en garantissant l'accessibilité et la flexibilité à tout moment.

W3C

Le W3C (World Wide Web Consortium) est une organisation internationale qui développe des standards et des recommandations pour assurer la croissance du web de manière cohérente et accessible. Fondée par Tim Berners-Lee, le W3C vise à promouvoir des technologies web ouvertes et à garantir l'interopérabilité des sites et applications sur différentes plateformes.

Partie 2 - Distinguez les langages du Web

Langages client

Les langages client sont des langages de programmation exécutés directement sur le navigateur de l'utilisateur, permettant d'interagir avec la page web sans nécessiter une communication constante avec le serveur. Les langages client les plus courants sont le HTML, le CSS et le JavaScript, utilisés pour créer et modifier le contenu, la structure et le comportement des pages web.

Navigateurs

Les navigateurs sont des logiciels qui permettent aux utilisateurs d'accéder, d'afficher et d'interagir avec des pages web sur internet. Parmi les navigateurs les plus populaires figurent Google Chrome, Mozilla Firefox, Safari et Microsoft Edge, qui interprètent le code des sites et affichent le contenu de manière compréhensible.

Langages serveur

Les langages serveur sont des langages de programmation exécutés sur un serveur web pour traiter les requêtes des utilisateurs et générer dynamiquement des pages ou des données. Parmi les langages serveur courants figurent PHP, Python, Ruby, Java, et Node.js, qui permettent de gérer la logique d'application, la base de données, et les interactions avec le client.

Frameworks

Les frameworks sont des ensembles d'outils et de bibliothèques qui facilitent le développement d'applications en fournissant une structure préétablie et des solutions pour des tâches courantes. Ils permettent aux développeurs de gagner du temps en simplifiant le code, en améliorant la sécurité et en offrant des bonnes pratiques pour construire des applications robustes et maintenables.

Bases de données

Les bases de données sont des systèmes de gestion qui permettent de stocker, organiser et récupérer efficacement des données de manière structurée. Elles peuvent être relationnelles, comme MySQL ou PostgreSQL, ou non relationnelles, comme MongoDB, et sont utilisées pour gérer de grandes quantités d'informations dans des applications, sites web et services en ligne.

Site responsive

Un site responsive est un site web conçu pour s'adapter automatiquement à la taille et à la résolution de l'écran, que ce soit sur un ordinateur de bureau, une tablette ou un smartphone. Cela garantit une expérience utilisateur optimale en ajustant la mise en page, les images et la navigation selon le dispositif utilisé.

Application mobile

Une application mobile est un logiciel conçu spécifiquement pour être installé et utilisé sur des appareils mobiles comme les smartphones et les tablettes. Elle peut offrir une gamme de fonctionnalités, allant de la gestion des tâches quotidiennes à l'accès à des services en ligne, en passant par le divertissement et la communication.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Les serveurs sont des ordinateurs ou des systèmes dédiés à fournir des services, des ressources ou des données à d'autres ordinateurs, appelés clients, via un réseau. Ils peuvent héberger des sites web, des applications, des bases de données ou gérer des courriels, et sont essentiels pour le bon fonctionnement de nombreux services en ligne.

Adresse IP

Une adresse IP (Internet Protocol) est un identifiant unique attribué à chaque appareil connecté à un réseau, permettant de localiser et de communiquer avec celui-ci sur internet ou un réseau local. Elle peut être de type IPv4 (composée de quatre groupes de chiffres) ou IPv6 (avec une structure plus longue et plus complexe), en fonction du protocole utilisé.

Nom d’hôte et DNS

Un nom d’hôte est l'étiquette attribuée à un appareil ou un serveur sur un réseau, permettant de l'identifier de manière lisible, comme "www.exemple.com". Le DNS (Domain Name System) est un système qui traduit ce nom d’hôte en une adresse IP, facilitant ainsi la navigation sur internet en remplaçant l'utilisation d'adresses numériques difficiles à retenir par des noms plus conviviaux.

Protocoles

Un protocole est un ensemble de règles et de conventions qui définissent la manière dont les données sont échangées entre des appareils ou systèmes sur un réseau. Il assure la communication efficace et sécurisée entre les différents éléments du réseau, comme le protocole HTTP pour la navigation web ou le protocole SMTP pour l'envoi d'emails.

Partie 4 - Identifiez les métiers du web

Développeurs

Les types de développeurs varient en fonction des technologies et des domaines dans lesquels ils se spécialisent. Un développeur front-end se concentre sur la partie visible des applications, en utilisant des langages comme HTML, CSS et JavaScript pour créer l'interface utilisateur. Un développeur back-end travaille sur la logique serveur, les bases de données et l'intégration des systèmes, utilisant des langages comme Python, Java ou PHP, tandis qu'un développeur full-stack maîtrise à la fois le développement front-end et back-end.

Product Managers et chefs de projets

Les Product Managers (PM) sont responsables de la stratégie et de la direction d'un produit, en définissant ses fonctionnalités et en coordonnant les efforts entre les équipes pour répondre aux besoins des utilisateurs. Ils établissent la feuille de route du produit et assurent sa réussite à long terme. En revanche, les chefs de projet se concentrent sur la gestion opérationnelle d'un projet spécifique, en supervisant la planification, l'exécution et la livraison tout en respectant les délais et les ressources allouées.

Designers

Un web designer combine souvent des compétences en UI (User Interface) et UX (User Experience), en créant à la fois l’aspect visuel : boutons, menus, typographies, couleurs ; et l'ergonomie des sites web : facilité de navigation, l'accessibilité et l'efficacité des interfaces, en prenant en compte les besoins et les attentes des utilisateurs. Il prend en charge la conception graphique globale et l'organisation du contenu.

Testeurs

Les testeurs dans le web, ou testeurs QA, sont chargés de vérifier la qualité d'un site ou d'une application avant sa mise en ligne. Ils effectuent des tests pour identifier les bugs, vérifier que les fonctionnalités fonctionnent correctement et assurer une bonne expérience utilisateur sur différents navigateurs et appareils. Leur rôle est crucial pour garantir que le produit final soit stable, performant et sans erreur.

Techniciens informatique

Les techniciens informatiques, membres de la grande famille des métiers des systèmes d'information (SI), sont responsables de la gestion du parc informatique, du support technique et des services réseau. Parmi eux, l'administrateur système veille à la configuration, à la maintenance et à la gestion des infrastructures informatiques pour assurer leur bon fonctionnement. Le responsable de la sécurité des systèmes d'information (RSSI) met en œuvre des stratégies de protection contre les cybermenaces, tandis que l'administrateur de base de données s'assure de la performance et de la disponibilité des bases de données essentielles à l'entreprise.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Le CSS (Cascading Style Sheets) est un langage de style utilisé pour définir l'apparence et la mise en page des éléments d'une page web, comme les couleurs, les polices, les espacements et les dispositions.

JavaScript

Front-End

Le JavaScript est un langage de programmation utilisé pour rendre les pages web interactives en permettant d'ajouter des fonctionnalités dynamiques, comme des animations, des formulaires interactifs ou des mises à jour en temps réel.

JavaScript

Back-End

Le JavaScript en back-end, généralement exécuté avec Node.js, permet de créer des applications serveur en utilisant le même langage que pour le front-end, offrant ainsi une solution unifiée pour le développement web complet.

PHP

Back-End

PHP est un langage de programmation côté serveur utilisé en back-end pour créer des sites web dynamiques, gérer les bases de données et traiter des requêtes HTTP, souvent en interaction avec des systèmes de gestion de contenu comme WordPress.

Java

Back-End

Java est un langage de programmation polyvalent, orienté objet, qui permet de créer des applications indépendantes de la plateforme, fonctionnant sur différents systèmes grâce à sa machine virtuelle (JVM). Il assure la logique métier dans des environnements d'entreprise à grande échelle.

Python

Back-End

Python est un langage de programmation simple et puissant, apprécié pour sa syntaxe claire et sa polyvalence, utilisé dans des domaines variés tels que le développement web, l'analyse de données, l'intelligence artificielle et l'automatisation.

C#

Back-End

C# est un langage de programmation orienté objet développé par Microsoft, principalement utilisé pour créer des applications sur la plateforme .NET, incluant des logiciels de bureau, des applications web et des jeux vidéo.

SQL

Bases de données

SQL (Structured Query Language) est un langage utilisé pour interagir avec des bases de données relationnelles, permettant de manipuler et de récupérer des données stockées sous forme de tables. Il permet aux utilisateurs d'effectuer des opérations telles que la création, la modification, la suppression et la consultation de données à l'aide de requêtes structurées.

Boîte à outils du développeur

  • IDE comme VS Code : Un IDE (Integrated Development Environment) comme VS Code est un logiciel qui offre un environnement complet pour écrire, tester et déboguer du code. Il inclut des fonctionnalités telles que l'édition de texte, la gestion de projets, l'intégration avec des systèmes de contrôle de version, et des outils de débogage pour faciliter le développement de logiciels.
  • Github : GitHub est une plateforme en ligne de gestion de code source qui utilise Git, permettant aux développeurs de collaborer, de stocker et de versionner leurs projets logiciels. Elle facilite le partage de code, la gestion des versions et le suivi des modifications grâce à des outils de collaboration comme les "pull requests" et les "issues".
  • Librairies / bibliothèques : Les librairies ou bibliothèques sont des ensembles de fonctions, de classes et de modules pré-écrits qui permettent aux développeurs de réutiliser du code pour accomplir des tâches courantes sans avoir à les coder de zéro. Elles facilitent le développement en offrant des solutions éprouvées pour des fonctionnalités comme la gestion des fichiers, la manipulation des données ou l'interface utilisateur.
  • Frameworks : Un framework est un ensemble d'outils, de bibliothèques et de conventions qui structure et facilite le développement d'applications. Il fournit une architecture de base pour le projet, permettant aux développeurs de se concentrer sur les fonctionnalités spécifiques sans avoir à redévelopper les composants de base.

    • Exemples de frameworks :

    1/ Python : Un framework web permettant de développer rapidement des applications web sécurisées et scalables en suivant le modèle MVC (Model-View-Controller).

    2/ React (JavaScript) : Un framework/bibliothèque utilisé pour créer des interfaces utilisateur dynamiques, principalement pour les applications web à une seule page.

    3/ Laravel (PHP) : Un framework PHP qui facilite le développement d'applications web en fournissant des outils puissants pour la gestion de bases de données, la sécurité et l'authentification.
